myFlix React

✦✦✦

Explore, Search, and Favorite Movies

Project Overview

The objective for this app was to build the client-side of the myFlix application using React, complementing the existing server-side (REST API and database). This single-page application (SPA) will allow movie enthusiasts to explore, search, and save information about their favorite films.

Technologies Used

  • React
  • React Bootstrap
  • Redux
  • Parcel
